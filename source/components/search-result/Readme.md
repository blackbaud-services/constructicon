### Examples

**Standard Use**

Provides a list item suited for a search results listing.

```
<SearchResult
  cta='Support'
  image='http://placehold.it/100x100'
  title='Support Name'
  subtitle='Supporter Charity'
  url='http://google.com'
  cta='Support'
/>
```

**Custom styles**

Apply a custom styles object to alter the look. Available elements are:

- `root`
- `avatar`
- `titles`
- `title`
- `subtitle`
